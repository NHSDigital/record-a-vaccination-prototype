What's new

Updates to Record a vaccination.

November 2025

Vaccinator role

  • In Manage users, when you add a user, you are now asked if the user is a vaccinator. This replaces a previous question that asked if the user was a registered clinician. Any user previously added to RAVS as a registered clinician will now appear as a vaccinator.
  • In Record vaccinations, when you select a vaccinator, the list of names is now populated with all users who have the role of vaccinator. If a vaccinator is missing from the list, make sure they are a vaccinator in Manage users.
  • In Find a patient, when you select a vaccinator, the list of names now only includes users who have the role of vaccinator. Previously it included all users. If a vaccinator is missing from the list, make sure they are a vaccinator in Manage users.

Eligibility and healthcare worker role

  • When you record the same vaccination for the next patient, you can now change the 'Eligibility' type (including 'Healthcare worker role') on the 'Check if these details still apply' page.

September 2025

New sections added

  • Record vaccinations is a new simpler way to record vaccinations. It will eventually replace the Find a patient section.
  • Recordsis a new section where you can view and edit records.
